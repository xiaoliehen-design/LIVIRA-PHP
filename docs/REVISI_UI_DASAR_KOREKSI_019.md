# Revisi UI Dasar Koreksi 019

Perubahan ini merapikan dropdown **Dasar koreksi** pada menu:

`Rekonsiliasi > Catat rekonsiliasi > Perubahan data barang`

## Perubahan

- Label dan dropdown ditata vertikal agar konsisten dengan field lain.
- Dropdown menggunakan lebar penuh pada kolomnya.
- Tinggi kontrol, radius sudut, border, bayangan, dan jarak dalam disamakan dengan gaya antarmuka LIVIRA.
- Ditambahkan tampilan hover, focus, dan disabled yang lebih jelas.
- Ikon panah dropdown dibuat konsisten dan tidak lagi menggunakan tampilan bawaan browser.
- Tata letak tetap responsif pada layar kecil.

## Database

Perubahan ini hanya menyentuh template HTML dan CSS. Tidak ada migration SQL baru yang perlu dijalankan.
